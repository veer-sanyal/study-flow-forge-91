# StudyFlow

**What should I practice next?**

A student learning app that ingests course materials, tracks mastery via spaced repetition, and generates daily study plans.

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (auth, database, edge functions, storage)
- ts-fsrs (spaced repetition)
- Vitest + Playwright

## Getting Started

```sh
npm install
npm run dev       # starts dev server on :8080
npm test          # run unit tests
npm run build     # production build
```
