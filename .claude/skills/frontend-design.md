# Frontend Design Skill (Study Flow Forge)
Typography:
- Prefer Inter (or existing font). Avoid system default.
- 14px body, 16px emphasis, 24–32px headings
- Line height: 1.5 body, 1.2 headings

Color:
- Purdue-ish: amber accents + slate neutrals
- Avoid pure black; use slate-900
- Ensure high contrast

Spacing:
- 4, 8, 12, 16, 24, 32, 48, 64
- Cards p-6, section gaps 8–12

Motion:
- Subtle hierarchy (expand/collapse, section transitions)
- No animation everywhere
